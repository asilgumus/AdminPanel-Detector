# Admin Panel Detector

A lightweight C# application to detect and enumerate administrative panels on websites.  
This tool helps security researchers, penetration testers, and web developers identify hidden or unsecured admin interfaces, providing quick and efficient scanning with customizable options.

## Features

- Scan websites for potential admin login pages
- Read User-Agent strings from a file
- Configurable scanning rules
- Reports detected panels in a simple, readable format
- Simple and easy-to-use C# console application

## Usage

1. Clone the repository
2. Add your target URLs to a text file
3. Optionally, add custom User-Agent strings
4. Run the application in a console


dotnet run
